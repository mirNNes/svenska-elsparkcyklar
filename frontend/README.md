# Frontend

Frontend-applikation för Svenska Elsparkcyklar.

## Installera beroenden

```bash
npm install
```

## Bygg och starta

```bash
npm run build   # Bygg frontend
npm start       # Starta lokal server (om inte Docker används)
```

Frontend körs på **port 3000**.

## Struktur

* `src/` - Källkod
* `dist/` - Byggd version
* `public/` - Statisk filservering
