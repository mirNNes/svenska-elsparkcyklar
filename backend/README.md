# Backend

Node.js/Express backend för Svenska Elsparkcyklar.

## Installera beroenden

```bash
npm install
```

## **Starta servern**

```bash
npm start
```

Servern körs på **port 5000**.

## Struktur

* `src/app.js` - Huvudapplikation
* `tests/` - Testfiler

## Tester
Kör tester:
PORT=3001 npm test

Kör tester med coverage:
PORT=3001 npm run test:coverage
